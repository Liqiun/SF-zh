# 软件基础

_[Software Foundations](https://softwarefoundations.cis.upenn.edu/)_ 中文版

**注意：正文编译自上游代码库；本代码库只接受 issue，PR 仅限于说明文档、测试脚本等。**

《软件基础》系列教材由 Benjamin C. Pierce 教授等多位作者合著。丛书分为《逻辑基础》、《程序语言基础》、《函数算法验证》、《QuickChick：软件特性测试》等卷，深入浅出地介绍了可靠软件的数理基础。

本书可供高年级本科生、研究生、科研工作者及具有同等学力的广大读者学习参考。阅读本书无需具备逻辑学、编程语言理论等背景知识，但一定的数学基础有助于理解书中内容。
